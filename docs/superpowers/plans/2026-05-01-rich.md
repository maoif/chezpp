# Rich Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Chezpp terminal rendering library modeled on Python Rich, using Scheme-style APIs, opaque styles, segment-based rendering, constructor macros, and a `(chezpp rich)` facade.

**Architecture:** Implement the library as focused Chez libraries under `chezpp/rich/`, with `(chezpp rich)` as the only supported user-facing import surface. The core pipeline is style object -> segment lines -> measurement/layout -> console encoder -> port output. Higher-level renderables, live display, progress, prompts, and pretty printing all use that shared pipeline.

**Tech Stack:** ChezScheme libraries, Chezpp `pcheck`, records, syntax-case macros, textual ports, deterministic test hooks, existing `tests/mat.sls` test harness.

---

## Source Design

Use the reviewed design in `docs/superpowers/specs/2026-05-01-rich-design.md`.

Important reviewed choices:

- Ignore previous local Rich implementation files.
- Constructor macros bind a variable name and use `:field value` clauses, for example `(rich-table t :title "Build" ...)`.
- Constructor macros only; mutators are procedures.
- `rich-style` accepts symbols and color forms, not style strings.
- `rich-style` returns an opaque style object; ANSI is emitted only by the console encoder.
- Later style options override earlier conflicting options.
- `rich-print` defaults to `(current-output-port)` when its first argument is not a console or output port.
- Reset is obtained with `(reset-style)`.
- If no terminal control API is available for no-echo password input, provide one in Chezpp rather than leaving password prompts unsupported.

  The `reset-style` procedure returns an opaque object that resets the active
  rich style when printed by rich output procedures.
  |#
  (define reset-style
    (lambda ()
      (make-rich-reset-record)))

  #|proc:rich-style
  The `rich-style` procedure constructs an opaque rich style object.
  |#
  (define rich-style
    (lambda args
      (make-rich-style-record args #f #f)))

  (define-record-type rich-console-record
    (fields (mutable output-port rich-console-output-port rich-console-output-port-set!)
            (mutable error-port rich-console-error-port rich-console-error-port-set!)
            (mutable input-port rich-console-input-port rich-console-input-port-set!)
            (mutable width rich-console-width rich-console-width-set!)
            (mutable color-system rich-console-color-system rich-console-color-system-set!)
            (mutable force-terminal? rich-console-force-terminal? rich-console-force-terminal?-set!)
            (mutable soft-wrap? rich-console-soft-wrap? rich-console-soft-wrap?-set!)
            (mutable theme rich-console-theme rich-console-theme-set!)
            (mutable ascii-only? rich-console-ascii-only? rich-console-ascii-only?-set!)))

  (define rich-console? rich-console-record?)

  #|proc:make-rich-console
  The `make-rich-console` procedure constructs a console using current ports and
  conservative terminal defaults.
  |#
  (define make-rich-console
    (case-lambda
      [()
       (make-rich-console-record (current-output-port)
                                 (current-error-port)
                                 (current-input-port)
                                 80
                                 'standard
                                 #f
                                 #f
                                 #f
                                 #f)]))

  #|macro:rich-console
  The `rich-console` macro constructs a console and binds it to an identifier.
  |#
  (define-syntax rich-console
    (lambda (stx)
      (syntax-case stx ()
        [(_ name)
         (identifier? #'name)
         #'(define name (make-rich-console))]
        [_ (syntax-error stx "invalid rich-console form")]))))

  (define $target+values
    (lambda (args)
      (cond [(null? args) (values (current-output-port) '())]
            [(or (output-port? (car args)) (rich-console? (car args)))
             (values (car args) (cdr args))]
            [else (values (current-output-port) args)])))

  (define $target-port
    (lambda (target)
      (if (rich-console? target)
          (rich-console-output-port target)
          target)))

  (define $write-rich-value
    (lambda (port value)
      (cond [(rich-style? value) (void)]
            [(rich-reset? value) (void)]
            [(string? value) (display value port)]
            [(char? value) (write-char value port)]
            [else (display value port)])))

  #|proc:rich-print
  The `rich-print` procedure prints values to a rich console, an output port, or
  the current output port.
  |#
  (define rich-print
    (lambda args
      (let-values ([(target values) ($target+values args)])
        (let ([port ($target-port target)])
          (pcheck ([output-port? port])
                  (for-each (lambda (value) ($write-rich-value port value)) values))))))

  #|proc:rich-println
  The `rich-println` procedure prints values and then writes a newline.
  |#
  (define rich-println
    (lambda args
      (let-values ([(target values) ($target+values args)])
        (let ([port ($target-port target)])
          (pcheck ([output-port? port])
                  (for-each (lambda (value) ($write-rich-value port value)) values)
                  (newline port))))))

  (define rich-fprint
    (lambda (port . values)
      (apply rich-print port values)))

  (define rich-fprintln
    (lambda (port . values)
      (apply rich-println port values)))

  (define rich-render
    (lambda (console value)
      (pcheck ([rich-console? console])
              value)))

  (define rich-export-text
    (lambda (value)
      (call-with-string-output-port
        (lambda (port)
          (rich-print port value)))))

  (define rich-export-ansi rich-export-text)

  (define rich-register-renderer!
    (lambda (pred renderer)
      (pcheck ([procedure? pred renderer])
              (void)))))

Create `chezpp/rich/private/common.ss`:

```scheme
#!chezscheme
(library (chezpp rich private common)
  (export rich-nonnegative-integer?
          rich-positive-integer?
          rich-rgb-integer?
          rich-output-target?
          rich-string-output
          rich-list-every?
          rich-list-any?
          rich-reverse-append)
  (import (chezpp chez)
          (chezpp utils))

  (define rich-nonnegative-integer?
    (lambda (x)
      (and (integer? x) (>= x 0))))

  (define rich-positive-integer?
    (lambda (x)
      (and (integer? x) (> x 0))))

  (define rich-rgb-integer?
    (lambda (x)
      (and (integer? x) (<= 0 x #xffffff))))

  (define rich-output-target?
    (lambda (x)
      (or (output-port? x) (record? x))))

  (define rich-string-output
    (lambda (proc)
      (let ([port (open-output-string)])
        (dynamic-wind
          void
          (lambda () (proc port))
          void)
        (get-output-string port))))

  (define rich-list-every?
    (lambda (pred xs)
      (let loop ([xs xs])
        (or (null? xs)
            (and (pred (car xs)) (loop (cdr xs)))))))

  (define rich-list-any?
    (lambda (pred xs)
      (let loop ([xs xs])
        (and (not (null? xs))
             (or (pred (car xs)) (loop (cdr xs)))))))

  (define rich-reverse-append
    (lambda (left right)
      (let loop ([left left] [right right])
        (if (null? left)
            right
            (loop (cdr left) (cons (car left) right)))))))
```

          rich-theme-set!)
  (import (chezpp chez)
          (chezpp utils)
          (chezpp rich private common))

  (define rich-ansi-reset "\033[0m")

  (define-record-type rich-style-record
    (fields (immutable fg rich-style-fg)
            (immutable bg rich-style-bg)
            (immutable bold? rich-style-bold?)
            (immutable dim? rich-style-dim?)
            (immutable italic? rich-style-italic?)
            (immutable underline? rich-style-underline?)
            (immutable blink? rich-style-blink?)
            (immutable reverse? rich-style-reverse?)
            (immutable hidden? rich-style-hidden?)
            (immutable strike? rich-style-strike?)))

  (define rich-style? rich-style-record?)

  (define-record-type rich-reset-record
    (fields))

  (define rich-reset? rich-reset-record?)

  (define-record-type rich-theme-record
    (fields (mutable table rich-theme-table rich-theme-table-set!)))

  (define rich-theme? rich-theme-record?)

  (define make-rich-theme
    (case-lambda
      [() (make-rich-theme-record (make-eq-hashtable))]
      [(alist)
       (pcheck ([list? alist])
               (let ([theme (make-rich-theme)])
                 (for-each
                  (lambda (entry)
                    (unless (and (pair? entry) (symbol? (car entry)) (rich-style? (cdr entry)))
                      (errorf 'make-rich-theme "invalid theme entry: ~a" entry))
                    (rich-theme-set! theme (car entry) (cdr entry)))
                  alist)
                 theme))]))

  #|macro:rich-theme
  The `rich-theme` macro constructs a theme from `:name style` clauses.
  |#
  (define-syntax rich-theme
    (lambda (stx)
      (syntax-case stx ()
        [(_ name) (identifier? #'name) #'(define name (make-rich-theme))]
        [(_ name :key value rest ...)
         (identifier? #'name)
         #'(begin
             (define name (make-rich-theme))
             (rich-theme-set! name 'key value)
             (rich-theme-add-clauses! name rest ...))]
        [_ (syntax-error stx "invalid rich-theme form")]))))

  (define-syntax rich-theme-add-clauses!
    (lambda (stx)
      (syntax-case stx ()
        [(_ theme) #'(void)]
        [(_ theme :key value rest ...)
         #'(begin
             (rich-theme-set! theme 'key value)
             (rich-theme-add-clauses! theme rest ...))]
        [_ (syntax-error stx "invalid rich-theme clause")]))))

  (define rich-theme-ref
    (lambda (theme name)
      (pcheck ([rich-theme? theme] [symbol? name])
              (hashtable-ref (rich-theme-table theme) name #f))))

  (define rich-theme-set!
    (lambda (theme name style)
      (pcheck ([rich-theme? theme] [symbol? name] [rich-style? style])
              (hashtable-set! (rich-theme-table theme) name style))))

  (define reset-style
    (lambda ()
      (make-rich-reset-record)))

  (define rich-color-system?
    (lambda (x)
      (memq x '(none standard 256 truecolor auto))))

  (define $named-fg-code
    (lambda (name)
      (case name
        [(black) 30] [(red) 31] [(green) 32] [(yellow) 33]
        [(blue) 34] [(magenta) 35] [(cyan) 36] [(white) 37]
        [(default) 39]
        [(bright-black) 90] [(bright-red) 91] [(bright-green) 92]
        [(bright-yellow) 93] [(bright-blue) 94] [(bright-magenta) 95]
        [(bright-cyan) 96] [(bright-white) 97]
        [else #f])))

  (define $named-bg-code
    (lambda (name)
      (case name
        [(on-black) 40] [(on-red) 41] [(on-green) 42] [(on-yellow) 43]
        [(on-blue) 44] [(on-magenta) 45] [(on-cyan) 46] [(on-white) 47]
        [(on-default) 49]
        [(on-bright-black) 100] [(on-bright-red) 101] [(on-bright-green) 102]
        [(on-bright-yellow) 103] [(on-bright-blue) 104]
        [(on-bright-magenta) 105] [(on-bright-cyan) 106]
        [(on-bright-white) 107]
        [else #f])))

  (define $rgb->ansi
    (lambda (prefix rgb)
      (let ([r (fxand (fxsra rgb 16) #xff)]
            [g (fxand (fxsra rgb 8) #xff)]
            [b (fxand rgb #xff)])
        (format "\033[~a;2;~a;~a;~am" prefix r g b))))

  (define $state
    (lambda ()
      (vector #f #f #f #f #f #f #f #f #f #f)))

  (define $state->style
    (lambda (s)
      (make-rich-style-record (vector-ref s 0)
                              (vector-ref s 1)
                              (vector-ref s 2)
                              (vector-ref s 3)
                              (vector-ref s 4)
                              (vector-ref s 5)
                              (vector-ref s 6)
                              (vector-ref s 7)
                              (vector-ref s 8)
                              (vector-ref s 9))))

  (define $apply-option!
    (lambda (s option)
      (cond
       [(symbol? option)
        (case option
          [(bold) (vector-set! s 2 #t)]
          [(no-bold) (vector-set! s 2 #f)]
          [(dim) (vector-set! s 3 #t)]
          [(no-dim) (vector-set! s 3 #f)]
          [(italic) (vector-set! s 4 #t)]
          [(no-italic) (vector-set! s 4 #f)]
          [(underline) (vector-set! s 5 #t)]
          [(no-underline) (vector-set! s 5 #f)]
          [(blink) (vector-set! s 6 #t)]
          [(no-blink) (vector-set! s 6 #f)]
          [(reverse) (vector-set! s 7 #t)]
          [(no-reverse) (vector-set! s 7 #f)]
          [(hidden) (vector-set! s 8 #t)]
          [(no-hidden) (vector-set! s 8 #f)]
          [(strike) (vector-set! s 9 #t)]
          [(no-strike) (vector-set! s 9 #f)]
          [else
           (let ([fg ($named-fg-code option)] [bg ($named-bg-code option)])
             (cond [fg (vector-set! s 0 option)]
                   [bg (vector-set! s 1 option)]
                   [else (errorf 'rich-style "invalid style option: ~a" option)]))])]
       [(rich-rgb-integer? option) (vector-set! s 0 option)]
       [(and (pair? option) (eq? (car option) 'fg) (pair? (cdr option)) (null? (cddr option)) (rich-rgb-integer? (cadr option)))
        (vector-set! s 0 (cadr option))]
       [(and (pair? option) (eq? (car option) 'bg) (pair? (cdr option)) (null? (cddr option)) (rich-rgb-integer? (cadr option)))
        (vector-set! s 1 (cadr option))]
       [else (errorf 'rich-style "invalid style option: ~a" option)])))

  #|proc:rich-style
  The `rich-style` procedure constructs an opaque style object from symbols and
  RGB color forms. Later options override earlier conflicting options.
  |#
  (define rich-style
    (lambda options
      (let ([s ($state)])
        (for-each (lambda (option) ($apply-option! s option)) options)
        ($state->style s))))

  (define $append-code
    (lambda (out code)
      (if code (cons (number->string code) out) out)))

  #|proc:rich-style->ansi
  The `rich-style->ansi` procedure encodes a style object as ANSI SGR bytes.
  |#
  (define rich-style->ansi
    (lambda (theme style)
      (pcheck ([rich-style? style])
              (let ([chunks '()])
                (when (rich-style-bold? style) (set! chunks (cons "1" chunks)))
                (when (rich-style-dim? style) (set! chunks (cons "2" chunks)))
                (when (rich-style-italic? style) (set! chunks (cons "3" chunks)))
                (when (rich-style-underline? style) (set! chunks (cons "4" chunks)))
                (when (rich-style-blink? style) (set! chunks (cons "5" chunks)))
                (when (rich-style-reverse? style) (set! chunks (cons "7" chunks)))
                (when (rich-style-hidden? style) (set! chunks (cons "8" chunks)))
                (when (rich-style-strike? style) (set! chunks (cons "9" chunks)))
                (let ([fg (rich-style-fg style)] [bg (rich-style-bg style)])
                  (cond [(integer? fg) (set! chunks (cons (substring ($rgb->ansi 38 fg) 2 (fx- (string-length ($rgb->ansi 38 fg)) 1)) chunks))]
                        [fg (set! chunks ($append-code chunks ($named-fg-code fg)))])
                  (cond [(integer? bg) (set! chunks (cons (substring ($rgb->ansi 48 bg) 2 (fx- (string-length ($rgb->ansi 48 bg)) 1)) chunks))]
                        [bg (set! chunks ($append-code chunks ($named-bg-code bg)))])

(library (chezpp rich segment)
  (export rich-segment
          make-rich-segment
          rich-segment?

          rich-segment-control?
          rich-segment-width

          rich-segment-pad-right)

          (chezpp utils)
          (chezpp rich style)
          (chezpp rich private common))

  (define-record-type rich-segment-record
    (fields (immutable text rich-segment-text)
            (immutable style rich-segment-style)
            (immutable control? rich-segment-control?)))

  (define rich-segment? rich-segment-record?)

  (define make-rich-segment
    (case-lambda
      [(text) (make-rich-segment text #f #f)]
      [(text style) (make-rich-segment text style #f)]
      [(text style control?)
       (pcheck ([string? text] [boolean? control?])
               (when (and style (not (rich-style? style)) (not (rich-reset? style)))
                 (errorf 'make-rich-segment "invalid segment style: ~a" style))
               (make-rich-segment-record text style control?))]))

  #|proc:rich-segment
  The `rich-segment` procedure constructs a segment from text and optional style.
  |#
  (define rich-segment make-rich-segment)

  (define rich-strip-ansi
    (lambda (str)
      (pcheck ([string? str])
              (let ([out (open-output-string)])
                (let loop ([i 0] [escape? #f])
                  (cond [(fx= i (string-length str)) (get-output-string out)]
                        [escape?
                         (if (char=? (string-ref str i) #\m)
                             (loop (fx+ i 1) #f)
                             (loop (fx+ i 1) #t))]
                        [(and (char=? (string-ref str i) #\esc)
                              (fx< (fx+ i 1) (string-length str))
                              (char=? (string-ref str (fx+ i 1)) #\[))
                         (loop (fx+ i 2) #t)]
                        [else
                         (write-char (string-ref str i) out)
                         (loop (fx+ i 1) #f)]))))))

  (define $display-width
    (lambda (str)
      (string-length (rich-strip-ansi str))))

  (define rich-segment-width
    (lambda (seg)
      (pcheck ([rich-segment? seg])
              (if (rich-segment-control? seg)
                  0
                  ($display-width (rich-segment-text seg))))))

  (define rich-segments-width
    (lambda (segments)
      (pcheck ([list? segments])
              (let loop ([segments segments] [n 0])
                (if (null? segments)
                    n
                    (loop (cdr segments) (+ n (rich-segment-width (car segments)))))))))

  (define rich-segments->plain
    (lambda (segments)
      (pcheck ([list? segments])
              (apply string-append (map rich-segment-text segments)))))

  (define $substring-segment
    (lambda (seg start end)
      (make-rich-segment (substring (rich-segment-text seg) start end)
                         (rich-segment-style seg)
                         (rich-segment-control? seg))))

  (define rich-segment-crop
    (lambda (segments width)
      (pcheck ([list? segments] [positive-natural? width])
              (let loop ([segments segments] [left width] [out '()])
                (cond [(or (null? segments) (fx= left 0)) (reverse out)]
                      [else
                       (let* ([seg (car segments)] [w (rich-segment-width seg)])
                         (cond [(fx<= w left)
                                (loop (cdr segments) (fx- left w) (cons seg out))]
                               [else
                                (reverse (cons ($substring-segment seg 0 left) out))]))])))))

  (define rich-segment-wrap
    (lambda (segments width)
      (pcheck ([list? segments] [positive-natural? width])
              (let ([text (rich-segments->plain segments)])
                (let loop ([i 0] [out '()])
                  (if (fx>= i (string-length text))
                      (reverse out)
                      (let ([end (min (string-length text) (fx+ i width))])
                        (loop end (cons (list (rich-segment (substring text i end))) out)))))))))

  (define rich-segment-pad-right
    (lambda (segments width)
      (pcheck ([list? segments] [natural? width])
              (let ([current (rich-segments-width segments)])
                (if (fx>= current width)
                    segments
                    (append segments (list (rich-segment (make-string (fx- width current) #\space)))))))))
)
